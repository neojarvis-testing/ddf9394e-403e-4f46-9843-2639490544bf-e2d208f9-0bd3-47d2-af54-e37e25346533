Mentorship Program Application

The Mentorship Program Application system allows users to apply for mentorship, view registered details, and manage their profiles. It supports multiple user roles such as Admin and User. Users can register, login, and access their personalized dashboards. The SPA is built using Angular and includes routing, services, and modular components.
